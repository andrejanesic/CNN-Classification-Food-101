# CNN Classification of Food 101 Dataset

**[This Jupyter notebook](./main.ipynb)** shows the process of developing and iteratively testing a convolutional neural network to classify 101 types of food.

**NOTE:** The work is based on other notebooks found on Kaggle. Data courtesy of [ETH Zurich.](http://data.vision.ee.ethz.ch/cvl/food-101.tar.gz)
